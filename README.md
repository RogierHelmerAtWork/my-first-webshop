# My First Webshop

A simple, responsive e-commerce website built with HTML, CSS, and JavaScript.

## Features

- 🛍️ Product catalog with sample items
- 🛒 Shopping cart functionality
- ➕ Add/remove items from cart
- 🔢 Quantity management
- 💰 Real-time price calculation
- 📱 Responsive design

## Getting Started

### Prerequisites

- Node.js (for running the development server)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/RogierHelmerAtWork/my-first-webshop.git
cd my-first-webshop
```

2. Install dependencies:
```bash
npm install
```

### Running the Application

Start the development server:
```bash
npm start
```

Then open your browser and navigate to `http://localhost:8080`

Alternatively, you can simply open the `index.html` file directly in your web browser.

## Project Structure

```
my-first-webshop/
├── index.html      # Main HTML file
├── styles.css      # CSS styles
├── script.js       # JavaScript functionality
├── package.json    # Node.js dependencies
└── README.md       # This file
```

## Usage

1. Browse products on the home page
2. Click "Add to Cart" to add items to your shopping cart
3. Navigate to the Cart to view your items
4. Adjust quantities using +/- buttons
5. Click "Checkout" to complete your order (demo only)

## Customization

You can easily customize this webshop by:

- **Adding products**: Edit the `products` array in `script.js`
- **Changing styles**: Modify `styles.css`
- **Adding features**: Extend functionality in `script.js`
